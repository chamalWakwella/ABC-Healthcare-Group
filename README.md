# Hospital Management System

A complete hospital management system built with Next.js 16, SQLite, and TypeScript.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to access the application.

## Default Admin Account

- **Email**: `admin@hospital.com`
- **Username**: `admin`
- **Password**: `admin123`

**Important**: Change the default admin password after first login!

## Features

Role-based access control (Admin, Doctor, Nurse, Radiologist, Patient), patient management, staff management, assignment system, prescriptions, radiology reports, and notifications.

## Technology Stack

Next.js 16, SQLite, TypeScript, Tailwind CSS
